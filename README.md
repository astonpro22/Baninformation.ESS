    <ul>
      <li><span class="highlight">Exploiting</span> or attempting to modify the game.</li>
      <li><span class="highlight">Doxxing</span> or sharing private information.</li>
      <li>These bans are typically <span class="highlight">permanent</span>.</li>

    <h3>Types of Discord Bans</h3>
    <ul>
      <li><span class="highlight">Temporary bans</span> for minor disruptions.</li>
      <li><span class="highlight">Permanent bans</span> for severe or repeated violations.</li>

      <li><span class="highlight">Doxxing</span> or sharing harmful links.</li>
